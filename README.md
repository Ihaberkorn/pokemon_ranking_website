# Pokémon Tier List Web App

**Live Demo:** [https://your-app.onrender.com  ](https://pokemon-ranking-website.onrender.com)
**Source:** This repository

A full-stack Flask web application that allows users to rank Pokémon by generation using an interactive tier-list system. Users assign 1–5 star ratings and see rankings update dynamically in the UI.

Built to demonstrate clean backend architecture, responsive frontend behavior, and thoughtful UI design around a familiar domain.

---

## Features
- Interactive tier-based ranking system  
- Generation-based navigation  
- Persistent rankings (saved locally)  
- Drag-and-adjust Pokémon within tiers  
- Real-time UI updates without full page reloads  

---

## Tech Stack
- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite  
- **Deployment:** Render  

---

## What This Project Shows
- REST-style routing and server-side rendering with Flask  
- Separation of concerns between backend logic and frontend interactivity  
- Client-side state management and DOM manipulation  
- Data persistence and dynamic content rendering  

This project was built as a complete, working application—not a static demo—and reflects practical full-stack development practices.
